This is a RestApi implementet with use of the Spring Boot just for practice.
The endpoint localhost:8080/calculate/<operation>?a=<double1>&b=<double2> will return a JSON with the result 
of the operation specified in the path variable <operation> (for <operation> one should substitute add, substract, multiply or divide).
For <double1> and <double2> one should substitue numeric values for the calculation.
